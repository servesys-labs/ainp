# Quick Submit Guide - IETF Draft

## ⚡ Fast Track (5 minutes)

### Step 1: Convert to v3 Format

1. **Go to**: https://xml2rfc.tools.ietf.org/
2. **Upload**: `docs/rfcs/draft-ainp-protocol-00.xml`
3. **Download**: Converted v3 XML file
4. **Save as**: `draft-ainp-protocol-00-v3.xml`

### Step 2: Submit

1. **Login**: https://datatracker.ietf.org/submit/
2. **Upload**: `draft-ainp-protocol-00-v3.xml` (the v3 file!)
3. **Fill form**:
   - Title: "AI-Native Network Protocol (AINP) for Semantic Agent Communication"
   - Abstract: Copy from draft
   - Status: "Standards Track" or "Informational"
   - Stream: "Independent Submission"
   - Author: Eswara Prasad Nagulapalli, contact@servsys.com, Servesys Labs
4. **Submit**

### Done! ✅

Your draft will be assigned a name like `draft-nagulapalli-ainp-protocol-00` and be available publicly within 24 hours.
